# push_swap Project Documentation

## Introduction

The push_swap project is a coding challenge that involves creating an efficient sorting algorithm for a stack-based data structure. The challenge requires participants to sort a list of integers using a limited set of operations on two stacks, `a` and `b`.

## Objectives

- Implement efficient sorting algorithms.
- Use a minimal number of operations to sort the stacks.
- Understand the principles of stack data structures and algorithms.

## Structure

The project consists of several key components:

- The main program that manages the stacks and operations.
- Algorithms for sorting the stack efficiently.
- Documentation outlining the implementation and usage.

## Getting Started

To get started with the push_swap project:
1. Clone the repository from GitHub.
2. Compile the source code.
3. Run the main program with a list of integers to sort.

## Conclusion

The push_swap project is an excellent opportunity to improve your understanding of stack algorithms and data structures. By following this documentation, you can implement your own solution and enhance your programming skills in C.
